FormSim.HTTPHandler Class Reference

The HTTPHandler class is designed to handle FRCs over the HTTP protocol. Since this protocol uses key value pairs, it is important that the passed arguments be formatted and parsed correctly. This implementation handles both of those scenarios. See GenericHandler for the best implementation strategy of dealing with FRCs in general. More...
