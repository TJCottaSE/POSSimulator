Detailed Description

The RestHandler class is designed to handle FRC's over the HTTP protocol to the RESTful API. This implementation uses JSON as the payload format, so serializing and deserializing are handled here. It also acts as a translation layer between the required Dictionary of strings, and the returned JSON from the REST interface.
